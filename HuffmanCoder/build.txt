Сборка приложения

Требования
- Установлена JDK 

Сборка
- Откройте командную строку или терминал
- Перейдите в каталог, в который вы хотите склонировать репозиторий
- Клонируйте репозиторий с помощью команды Git:
  git clone https://github.com/sophy-korp/Java_Labs_5term.git
- Перейдите в директорию HuffmanCoder
- Скомпилируйте все файлы .java:
  javac *.java
- Проверьте, что появились файлы .class

Упаковка в JAR-файл (опционально)
- Создайте файл Manifest.txt с содержимым
  Main-Class: Main
- Упакуйте проект в JAR-файл
  jar cfm ProjectName.jar Manifest.txt *.class
